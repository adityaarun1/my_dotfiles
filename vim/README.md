# VIM configurations

Requirement `vim > 0.8`

Install [vim-plug](https://github.com/junegunn/vim-plug)

Install [coc.nvim](https://github.com/neoclide/coc.nvim) for auto suggestions.

Directory Structure

`$HOME/.vim` <br>
&nbsp;&nbsp;|-- `autoload` <br>
&nbsp;&nbsp;|-- `backup` <br>
&nbsp;&nbsp;|-- `colors` <br>
&nbsp;&nbsp;|-- `config` > `coc.vim` <br>
&nbsp;&nbsp;|-- `plugged` <br>
